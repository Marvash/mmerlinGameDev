Minecraft clone

In this project I reproduced some of the main features of Minecraft using only c++ and opengl. The goal of this project was not to remake the game but to make a functioning game prototype from scratch with no game engines or frameworks while also improving my opengl and c++ skills and knowledge. Minecraft just happened to be a good candidate as the voxel world simplifies a lot of things. Working to this project allowed me to learn, study and experiment with many game dev related concepts like programming patterns in the context of games, procedural generation algorithms, game physics with fixed timestep, game engine architectures and mesh generation to name a few. The main features of this prototype are multithreaded procedural terrain generation, procedural biomes, block interaction (breaking and placing), crafting, tools and three different movement modes (Normal, Fly, Fly no collisions).

Space Miner (W.I.P.)

Space Miner is a space themed procedural 2D action game made with Unity. The objective of the game is to defeat the boss of each level. To do so, the player must acquire stronger and stronger weapons by collecting and selling minerals found in asteroids spread around the level. The player always spawns near its base at the center of the level and can fly outward in any direction. The further the player goes from the base the stronger are the enemies he must fight and the more rare are the minerals he can find in the asteroids. This game is still a work in progress and only few of the mechanics described above are implemented. The main features currently implemented are asteroid mining, mineral random drop and collection, mineral selling, procedural asteroid distribution, progressively harder enemies and multiple weapons.
